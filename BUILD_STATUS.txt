  核心算法移植状态

✅ EncodingDetector.detectEncoding()
   - 从 C# 的 DetectEncoding() 移植
   - 支持 Shift-JIS/UTF-8/UTF-16
   - 算法完全一致

✅ ImageProcessor.applyOpacity()
   - 从 C# 的 GetTransparent() 移植
   - 使用 int[] pixels 替代 unsafe 指针
   - 算法等价

✅ ImageProcessor.calculateAlphaSum()
   - 从 C# 的 AddAlphas() 移植
   - 用于图层排序
   - 算法完全一致

✅ ImageProcessor.mergeLayers()
   - 从 C# 的 Layer.GenerateImage() 移植
   - 使用 Canvas 替代 Graphics
   - 算法逻辑一致

✅ CoordinateFileParser.parseTxtFile()
   - 从 C# 的 TXTInitialize() 移植
   - Tab 分隔格式解析
   - 逻辑完全一致

✅ CoordinateFileParser.parseJsonFile()
   - 从 C# 的 JSONInitialize() 移植
   - 使用 Gson 替代正则表达式
   - 更简洁易维护

================================================================================
  构建方法
================================================================================

方法 1: Android Studio (推荐)
  1. 用 Android Studio 打开项目
  2. Build > Build Bundle(s) / APK(s) > Build APK(s)
  3. 从 app/build/outputs/apk/debug/ 获取 APK

方法 2: GitHub Actions (自动化)
  1. 推送到 GitHub 仓库
  2. GitHub Actions 自动触发构建
  3. 从 Actions 页面下载构建产物

方法 3: 命令行 (需要真实 Linux)
  ./gradlew assembleDebug

⚠️  PRoot/Termux 环境限制:
  当前环境无法运行 AAPT2，请使用上述方法之一
